Finger Painting


Website Published: April 2025

In this project I utilized the ml5.js library, more specifically its HandPose model figure, and incorporated it into a P5.js code which allows you to finger paint via camera tracking

My idea was to track all five fingers and utilize it to create visuals based on fingerspelling in ASL. For instance, the image you clicked on to get here was created when I fingerspelled my name (Samantha).
